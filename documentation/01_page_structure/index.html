Content Structure

This section is extremly important to understand the functionality of the content, so please read it carefuly! 

<div class="all-elements">
    <div id="sidebar" class="page-sidebar">
        <div class="page-sidebar-scroll">
        <!-- Explained in Navigation Stucture-->
        </div>
    </div>
    <div id="content" class="page-content">
        <div class="content">
            <!--(1)-->
        <div>

        <div class="content">
            <div class="container">
                <!---(2)---->
            </div>
        </div>

    </div>
<div>

(1) the content class creates a margin left and right, to give your content a bleed margin, inside the content, you can add containers (2) containers will house all the elements found in the second documentation exept for the images sliders which will be added directly in the content (1) class.

(1) - The top part of the sidebar, the 5 icons at the top, these can be linked to any page you want or used to close the navigation, adding the sidebar-close class will make the button close the sidebar

(2) - The sidebar header is the part with the rounded logo and the text. You can remove the text if you wish and simply add a png logo of your choosing, just be sure to open style.css and locate the sidebar-logo class and give it the width of your logo so the image is centered! 

(3) - The sidebar breadcrumb is the darker line with the text, this can be added after every element you wish! 

(4) - The nav item in 4 will create a simple navigation item, like the Home button in the sidebar in the demo files. Adding the em class selected-nav will make the green ball appear to the right, if you wish the ball to be grey(unselected) simply give it the class unselected-nav. 

(5) The nav item with a submenu attached to it. Cop paste the entire lines from 33 to 39 ( in the documentation code snip ) to create the navigation with a submenu. Duplicate line 37 as many times as needed to create more submenus,. To make a submenu item selected, simply give the em the class selected-sub-nav. 

To make the menu with a deployed ( active ) submenu, copy and paste the code from lines 40 to 46 and duplicate line 44 to create more subnav items! 

(6) The sidebar updates are simply there as a notification, you can add how many you want and the variation colors are red-update, blue-update, yellow-update and orange-update. 

The sidebar-decoration class will create a horizontal line! 

 

 

 

 

 

 

 

Creating Background Image Sections

The background image section spans the full width of the page, so when creating one you must add it outside of the content class (NOT the content ID, the content ID houses the entire page content, so for example:

 

<div class="content">
    <div class="decoration"></div>
</div>

<!--(1)-->
<div class="content-section">
    <h1 class="left-text">INTRODUCTION</h1>
    <em class="left-text">What is Photroller?</em>
    <div class="overlay"></div>
    <img class="responsive-image" src="images/general-nature/5w.jpg" alt="img">
</div>

<div class="content">
    <div class="decoration"></div>
    <div class="container no-bottom">
        <p>
          More Content
        </p>
    </div>
</div>

(1) copying the content-section class and it's content will create a fullwidth image that you can use as a section 

Slider Navigation or Solid Navigation

Solid Navigation

To create a solid navigation, in the page-content class or content id, right after opening it add the following 

<div class="content-controls solid-color fixed-header">
    <a href="#" class="deploy-sidebar"></a>
    <em class="content-title">MEET PHOTROLLER</em>
    <a href="contact.html" class="deploy-contact"></a>
</div>    
<div class="fixed-header-clear"></div>

This will generate a solid color header, 50 pixels high, you can change the color variations by replacing the stylesheet found on every page at line 16 from blue.css to black.css , green.css, magenta.css, orange.css or red.css

Slider Navigation

To create a slider navigation for your pages, in the page-content class or content id class, right after opening it, add the folowing.

<!---(1)--->
<div class="content-controls">
    <a href="#" class="deploy-sidebar"></a>
    <em class="content-title">Welcome</em>
    <a href="contact.html" class="deploy-contact"></a>
</div>

<div class="homepage-slider">
    <div class="slider-controls" data-snap-ignore="true">     
        <!---(2)--->           
        <div>
            <div class="overlay"></div>
            <img src="1.jpg" class="responsive-image" alt="img">
        </div>
        <div>
            <div class="overlay"></div>
            <img src="2.jpg" class="responsive-image" alt="img">
        </div>
        <div>
            <div class="overlay"></div>
            <img src="3.jpg" class="responsive-image" alt="img">
        </div>
    </div>
    <!--(3)--->
    <div class="content-header">
        <h3>PHOTROLER</h3>
        <em>Simplicity always rocks!</em>
        <a href="tel: +1234567890" class="content-header-call"></a>
        <a href="sms: +1234567890" class="content-header-text"></a>
    </div>
</div>

(1) The content controls are simply the page title, the deploy and contact elements

(2) To add more images, simply duplicate the div with no class, the one containing the overlay and the image, the overlay opacity can be controled by editing style.css and searching for the .overlay class. replace the last value 0.x with your number 0.1 being the smallest 1 being the highest. 

(3) The content header is the text, subtitle and call and text buttons! It's highly recommended that you do not use an image for a logo over the slider as it might end up corupting the slider on certain devices! Use logos on pages with solid navigations. 

 

Using the Widefolio

The wide folio acts like an accordion! The structure is explained bellow, as for content? Go wild! You can basically add anything to them (except for image sliders and contact forms ) 

<div class="wide-folio">
    <div class="wide-item">
        <div class="wide-item-titles">
            <h4>A simple image</h4>
            <p>This image has no content when tapping it!</p>
        </div>
        <a class="wide-image" href="#">
           <div class="overlay"></div>
           <img class="responsive-image" src="images/general-nature/7w.jpg" alt="img">
        </a>
        <div class="wide-item-content">
           <!-- Add your content here, if the item expands to fullwidth, add it inside a <div class="container"></div> -->
        </div>
    </div>
</div>
